# DEF Aberturas

Sitio web oficial de **DEF Aberturas**, empresa especializada en la fabricación de puertas, ventanas, portones y aberturas de madera a medida.

## 🌐 Demo

🔗 https://defaberturas.com

---

# ✨ Características

* Diseño moderno y responsive
* Compatible con celulares, tablets y desktop
* Galerías dinámicas de:

  * Puertas
  * Ventanas
  * Portones
* Filtros por categorías y subcategorías
* Modal interactivo para imágenes
* Navegación táctil (swipe) en móviles
* Integración con WhatsApp
* Slider animado en sección “Nosotros”
* Optimización visual y rendimiento

# 🚀 Funcionalidades destacadas

## 🖼️ Galerías dinámicas

Las galerías funcionan mediante arrays en JavaScript:

```js
const puertaImages = []
const ventanaImages = []
const portonImages = []
```

Esto permite:

* agregar imágenes fácilmente
* aplicar filtros dinámicos
* abrir modal automático
* navegación entre imágenes
